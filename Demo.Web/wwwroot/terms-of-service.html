Gebruiksvoorwaarden

Laatste update: 15 januari 2024 | Versie 1.0
Door deze API te gebruiken gaat u akkoord met deze voorwaarden.

1. De Service

De Bookstore API biedt order management, klantenbeheer en voorraadmanagement voor boekverkoop.

2. API Key

  • U ontvangt een unieke API Key (X-API-Key)
  • Houd uw API Key geheim
  • Maximum 100 requests per minuut
  • Bij misbruik kunnen we uw toegang blokkeren

3. Toegestaan Gebruik

Toegestaan:

  • Orders plaatsen voor klanten
  • Klanten- en voorraadmanagement
  • Integratie met uw eigen systemen

Verboden:

  • Misbruik, hacking of reverse engineering
  • Doorverkoop van toegang
  • Spam of ongeautoriseerde marketing

4. Uw Data

  • U behoudt eigendom van uw klantgegevens
  • Wij voldoen aan AVG/GDPR
  • U kunt uw data altijd exporteren
  • Zie onze Privacy Policy voor details

5. Beschikbaarheid

  • Wij streven naar 99.5% uptime
  • Onderhoud wordt vooraf aangekondigd
  • Service wordt geleverd "as is"
  • Support: support@bookstore@ehb.be

6. Aansprakelijkheid

  • Maximale aansprakelijkheid beperkt tot abonnementsgeld
  • Geen aansprakelijkheid voor indirecte schade
  • Maak altijd backups van uw data

7. Wijzigingen & Opzegging

  • Wij kunnen deze voorwaarden wijzigen (30 dagen vooraf melding)
  • U kunt opzeggen met 1 maand opzegtermijn
  • Wij kunnen uw toegang beëindigen bij misbruik

8. Toepasselijk Recht

Belgisch recht is van toepassing. Geschillen: Rechtbank Brussel.
